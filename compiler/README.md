# Hasky Compiler

Hasky Compiler helps you transform high-level Hasky source code into assembly
instructions.

> **NOTE:** Hasky uses its own assembly format that isn't compatible with x86 or
> ARM assembly.

![error](img/highlight.png)

## Usage

```bash
hasky-compile help      # Print a brief help message.
hasky-compile version   # Display current Hasky Compiler version.
hasky-compile SOURCE.ha # Compile one Hasky file.
```
